FAQ

Common Questions

Do I need to know how to code to use the database?
Not at all. We build everything using Google Sheets or Airtable as the interface — tools you can use like a spreadsheet. The automation and technical plumbing is handled by us entirely. You just see clean, organised data in a familiar interface.
Can the system handle large volumes of orders?
Yes. Google Sheets can handle thousands of rows efficiently with our structure. For businesses with 500+ daily orders, we recommend Airtable or a dedicated database (Supabase/Firebase) for better performance — we can build either.
Will it connect with my existing Facebook AI chatbot?
Yes — our database systems are designed to work hand-in-hand with our AI Automation systems. If you have an existing chatbot, we can integrate the database to capture its data. If you build the AI system with us, the database integration is built in from day one.
Can multiple team members access the dashboard?
Absolutely. Google Sheets and Airtable both support multiple users with different permission levels. You can give admins full access and delivery staff read-only access to relevant order data. We configure the permission structure during setup.
